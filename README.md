# 简介
Fedora下提取黑苹果基础文件，用法：
```
git clone https://gitee.com/Sirius2/Fedora-OSbase.git && cd Fedora-OSbase && sudo chmod +x setup.sh &&./setup.sh
```

> 提取内容
+ DSDT
+ ALC info.

适用于macOS所有版本
